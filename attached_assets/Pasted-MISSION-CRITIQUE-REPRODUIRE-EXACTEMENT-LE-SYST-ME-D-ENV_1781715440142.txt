MISSION CRITIQUE — REPRODUIRE EXACTEMENT LE SYSTÈME D’ENVOI DE MÉDIAS BRUTEPAWA

Tu dois implémenter un système d’envoi de médias identique à la maquette de référence, avec un niveau de fidélité visuelle et fonctionnelle de 99%.

INTERDICTION DE SIMPLIFIER, REMPLACER OU INTERPRÉTER LE DESIGN.

══════════════════════════════
OBJECTIF
══════════════════════════════

Créer un système moderne de gestion des médias dans la messagerie BrutePawa :

- Photos
- Vidéos
- Documents
- Audio
- Localisation

Le comportement doit être similaire à WhatsApp, Telegram et Messenger 2026.

══════════════════════════════
ÉTAT 1 : ENVOI EN COURS
══════════════════════════════

Lorsqu'un utilisateur envoie un média :

✓ Afficher immédiatement la prévisualisation réelle du média

✓ Ne jamais afficher uniquement le nom du fichier

✓ Afficher l'image directement

✓ Afficher la miniature vidéo directement

✓ Afficher la carte de localisation directement

✓ Afficher le document dans une carte moderne

Ajouter en superposition :

- Cercle de progression
- Pourcentage réel
- Bouton Annuler (X)

Exemple :

68%
◯
 X

Le cercle doit se remplir progressivement.

Le pourcentage doit être calculé avec :

uploadedBytes / totalBytes

et non simulé.

══════════════════════════════
ÉTAT 2 : CONNEXION LENTE
══════════════════════════════

Si aucun progrès détecté pendant 3 secondes :

Afficher :

🕒 Connexion lente...

Le cercle reste visible.

Le pourcentage reste visible.

Le média reste visible.

Ne jamais remplacer le média.

══════════════════════════════
ÉTAT 3 : EN ATTENTE DU RÉSEAU
══════════════════════════════

Si internet est perdu :

Afficher :

🕒 En attente du réseau

Envoyé dès le retour de la connexion

Le média reste affiché.

Le cercle reste affiché.

Le téléchargement est suspendu.

Reprise automatique dès le retour du réseau.

══════════════════════════════
ÉTAT 4 : ÉCHEC D'ENVOI
══════════════════════════════

Si le serveur renvoie une erreur :

Afficher :

⚠ Échec de l'envoi

↻ Réessayer

Le média reste affiché.

Aucune suppression automatique.

Le bouton Réessayer relance l'envoi.

══════════════════════════════
ÉTAT 5 : ENVOYÉ
══════════════════════════════

Lorsque l'upload est terminé :

Supprimer :

- Cercle
- Pourcentage
- Loader

Afficher :

✓

Le média est présent sur le serveur.

Le destinataire ne l'a pas encore reçu.

══════════════════════════════
ÉTAT 6 : LIVRÉ
══════════════════════════════

Lorsque le destinataire reçoit le média :

Afficher :

✓✓

comme WhatsApp.
