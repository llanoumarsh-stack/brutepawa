RECHERCHE DE LANGUE

Les autres langues ne doivent PAS être affichées immédiatement.

Elles doivent être accessibles uniquement via la barre de recherche.

La recherche doit être :

- Temps réel
- Sans bouton rechercher
- Résultats instantanés
- Tolérante aux fautes de frappe
- Recherche par nom local
- Recherche par nom anglais
- Recherche par code ISO

Exemples :

"fr" → Français
"french" → Français
"français" → Français

"ar" → العربية
"arabic" → العربية

"zh" → 中文
"chinese" → 中文
