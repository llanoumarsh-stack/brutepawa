Problème :
Un appui long affiche actuellement les menus Android :

- Copier
- Partager
- Traduire
- Recherche Google
- Sélection de texte
- Poignées bleues

Ce comportement est interdit.

Comportement attendu :

✓ Aucun texte sélectionnable
✓ Aucun menu Android
✓ Aucun menu contextuel
✓ Aucun appui long sur publication
✓ Aucun appui long sur profil
✓ Aucun appui long sur commentaire
✓ Aucun appui long sur nom d'utilisateur
✓ Aucun déclenchement de Google

Appliquer globalement :

-webkit-user-select: none;
-webkit-touch-callout: none;

Bloquer également :

contextmenu
selectstart
longpress

sur toute l'application.
