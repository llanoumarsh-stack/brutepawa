API MODIFIER

Conditions :

- utilisateur connecté
- propriétaire du post

Mettre à jour :
- contenu
- médias
- updated_at

========================
API ÉPINGLER
========================

POST /api/posts/:id/pin

Logique :

1. Désépingler tous les autres posts du propriétaire

UPDATE posts
SET is_pinned = false
WHERE user_id = currentUserId;

2. Épingler le post sélectionné

UPDATE posts
SET is_pinned = true
WHERE id = postId;

Affichage du profil :

ORDER BY
is_pinned DESC,
created_at DESC

========================
API ARCHIVER
========================

POST /api/posts/:id/archive

UPDATE posts
SET is_archived = true
WHERE id = postId;

Les posts archivés ne doivent plus apparaître dans le fil.

Toutes les requêtes doivent filtrer :

WHERE is_archived = false

========================
API SUPPRIMER
========================

DELETE /api/posts/:id

Étapes :

1. Vérifier propriétaire
2. Supprimer fichiers Cloudflare R2
3. Supprimer likes
4. Supprimer commentaires
5. Supprimer notifications liées
6. Supprimer le post
