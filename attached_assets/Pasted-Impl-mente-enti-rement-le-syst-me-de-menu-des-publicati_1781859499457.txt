Implémente entièrement le système de menu (...) des publications BrutePawa avec frontend + backend + base de données + API + permissions + sécurité.

====================================================
FRONTEND
====================================================

Lorsqu'un utilisateur appuie sur les trois points (...) d'une publication :

Si la publication appartient à un autre utilisateur :

📌 Enregistrer la publication
🔗 Copier le lien
📤 Partager
🔔 Activer/Désactiver les notifications
🙈 Masquer cette publication
👤 Ne plus suivre l'auteur
⚠️ Signaler la publication
🚫 Bloquer l'utilisateur

Si la publication appartient à l'utilisateur connecté :

✏️ Modifier
📌 Épingler au profil
🗂️ Archiver
💬 Désactiver les commentaires
👁️ Modifier l'audience
📊 Voir statistiques
📤 Partager
🗑️ Supprimer

Bottom Sheet premium :
- Coins 24px
- Animation fluide
- Fermeture par swipe
- Fermeture en cliquant hors zone
- Compatible Android/iOS
- Design moderne type Facebook / Threads
