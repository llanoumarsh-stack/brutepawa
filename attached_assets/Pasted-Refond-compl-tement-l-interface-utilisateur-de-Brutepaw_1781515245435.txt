Refond complètement l'interface utilisateur de Brutepawa pour lui donner une apparence moderne, premium, professionnelle et comparable aux grandes applications sociales actuelles.

OBJECTIF

Conserver toutes les fonctionnalités existantes mais améliorer fortement le design, l'expérience utilisateur et la perception de qualité.

====================================================
DESIGN GÉNÉRAL
====================================================

- Utiliser un design moderne 2026.
- Coins arrondis harmonieux.
- Ombres légères et élégantes.
- Espacements cohérents.
- Animations fluides.
- Respecter les couleurs principales de Brutepawa.
- Donner une impression premium dès l'ouverture.

====================================================
PAGE MESSAGES
====================================================

Refaire complètement l'écran Messages.

Actuellement l'écran paraît vide.

Créer un header moderne :

Messages

Afficher :

- Nombre de messages non lus
- Recherche moderne
- Nouveau message
- Menu

Ajouter sous le header :

- Discussions récentes
- Groupes récents
- Canaux récents

Supprimer les grands espaces vides.

Ajouter des cartes modernes.

Menu animé :

Animation fluide.

Uniformiser les boutons.

Uniformiser les cartes.

====================================================

"Cette application est moderne, sérieuse et professionnelle."
